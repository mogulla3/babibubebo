# Babibubebo
`babibubebo` language converter

## Examples
```ruby
Babibubebo.encode("こんにちは")
# => "こぼんぶにびちびはば"

Babibubebo.decode("こぼんぶにびちびはば")
# => "こんにちは"
```

## Installation
```
gem 'babibubebo'
```

And then execute:

```
$ bundle
```

 Or install it yourself as:

```
$ gem install babibubebo
```
